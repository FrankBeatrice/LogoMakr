# Free Online LogoMakr

It is a desktop app of Free Online Logo maker https://logomakr.com/

## Setup

```bash
sudo npm install -g electron 
```
## Clone the Repo

```bash
git clone https://github.com/JeFcorp/LogoMakr.git
```

## Install Dependencies

```bash
cd LogoMakr
npm install
```

## Run App
```bash
./logo-maker
```
