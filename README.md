# Free Online LogoMaker

## Setup

```bash
sudo npm install -g electron 
```
## Clone the Repo

```bash
git clone https://github.com/JeFcorp/LogoMaker.git
```

## Install Dependicies

```bash
cd LogoMaker
npm install
```

## Run App
```bash
./logo-maker
```
